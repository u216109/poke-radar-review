Application Details

Poke Radar is used by our company as an internal decision-support tool for Pokemon trading card market analysis. It is not a public consumer marketplace, not an eBay listing tool, and not an application for collecting eBay usernames, passwords, buyer details, seller details, or private user information.

Required API Buy Marketplace Insights API
Required OAuth scope buy.marketplace.insights
Use case Internal market analytics and price-gap monitoring

Purpose of Request

We are requesting Production access to a restricted API: the Buy Marketplace Insights API.

The application compares recent eBay sold activity, transaction volume, and price trends with Japanese market prices from Pokeca Chart. The purpose is to identify cards with increased overseas demand or significant price gaps between overseas and Japanese markets.

The requested OAuth scope is https://api.ebay.com/oauth/api_scope/buy.marketplace.insights.

How the Application Works

1. Candidate discovery Select monitored Pokemon TCG cards from Japanese market activity and eBay activity signals.
2. Market data collection Collect recent eBay sold activity and Japanese market prices for monitored cards.
3. Aggregated analysis Calculate price, transaction count, price gap, gap rate, and ranking score.
4. Internal reporting Display results in a local dashboard and internal reports for business analysis.

Data Usage and Compliance

What we use

  • Aggregated sold activity and pricing signals.
  • Recent transaction count and price trend indicators.
  • Comparison with Japanese market prices from Pokeca Chart.
  • Internal ranking and monitoring reports.

What we do not do

  • We do not resell raw eBay data.
  • We do not display buyer or seller private information.
  • We do not collect eBay usernames or passwords.
  • We do not share raw eBay data with third parties.

Operational Details

  • The application runs scheduled analysis approximately once per day.
  • Expected call volume is about 100 to 300 API calls per day, with a maximum estimated peak of up to 1,000 calls per day during testing or expanded monitoring.
  • The application does not perform continuous polling.
  • OAuth credentials are stored server-side only.
  • Secrets are not exposed in logs, browser UI, or client-side code.
  • The application handles API errors gracefully and limits API calls to the minimum needed for analysis.

Public Availability

The operational analytics tool is an internal local web application used by our company only and is not publicly accessible. This page is provided as a public information page for eBay application review.
